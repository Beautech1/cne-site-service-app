Manage Maps / Buildings
Add building floor plans or other map views. These are saved centrally in Supabase so all devices see the same map list. For floor plans, use an image path such as ./food-building-main.png or a Supabase Storage public URL.
Image size guidance
Normal floor plans: 2500–4000 px wide
Large detailed plans: 4000–6000 px wide
Try to keep each image under 5–8 MB for better phone/laptop performance.
Use JPG for smaller file size, or PNG for sharper line drawings.
Tip: If you add a new local image path, keep that image file in the same folder as this HTML file. If you use Supabase Storage, paste the public image URL here.
